イベントハンドラ

jQuery Mobile のイベントを簡便にハンドリングするAPIである。

幾つかの スコープ インスタンス のプロパティ名はイベントハンドラの為に予約される。

プロパティ名イベント
onPageInitページの初期化
onBeforeShowページ表示前
onShowページ表示
onBeforeHideページ非表示前
onHideページ非表示

これらの名前のプロパティが存在する場合、各々のイベントのタイミングで関数として呼び出される。 引数は元のイベントとそのスコープ要素の jQuery Object である。

デフォルトの置換属性の処理は onBeforeShow の直後に行われる。 つまりスコープに操作を加えるならばこのイベントハンドラで行うべきである。

使用例： 
<script>
  function PageDriver($this) {
    this.$this = $this;
  }
  PageDriver.prototype.onPageInit = function(ev,$this) {
    alert("page init");
  }
  PageDriver.prototype.onBeforeShow = function(ev,$this) {
    alert("before show");
  }
  PageDriver.prototype.onShow = function(ev,$this) {
    alert("show");
  }
  PageDriver.prototype.onBeforeHide = function(ev,$this) {
    alert("before hide");
  }
  PageDriver.prototype.onHide = function(ev,$this) {
    alert("hide");
  }
</script>

<div data-role="page" data-dp-scope="(new PageDriver($this))">
</div>
