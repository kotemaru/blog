Getting Started with JQMDP

前提条件として jQuery mobile を利用する準備が整っていなければならない。
jQuery mobile の準備については こちらへ(http://jquerymobile.com)

まずは jquery.mobile.jqmdp-1.0rc2.js をダウンロードして自分のWebサイトに配置する。
配置した jquery.mobile.jqmdp-1.0rc2.js を jquery.mobile-1.1.0.js の直前に読み込めば準備完了である。

<script src="jquery-1.7.1.js"></script> <script src="jquery.mobile.jqmdp-1.0rc2.js"></script> <script src="jquery.mobile-1.1.0.js"></script>

簡単な例として現在の日時を表示するページの記述を比較する。

jQuery Mobile: 

<script>
$(function(){
  $("#index").live('pagebeforeshow', function(ev) {
    $("#currentDate").text(""+new Date());
  });
})
</script>

<div>Date: <span id="currentDate"></span></div>

JQMDP: 

<div>Date: <span data-dp-text="(new Date())"></span></div>

実行結果はいずれも同じである。
Date:

このように JQMDP は data-dp- で始まるカスタム属性を利用し簡単に動的なページを作成できる。
